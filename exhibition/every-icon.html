Every Icon

John F. Simon Jr.

Year:1997

This piece is a program that generates every possible black and white image on a 32x32 pixel grid. Each square can only be black or white, but even with that simple setup, the number of possible combinations becomes insanely large, way beyond what the program can realistically finish. The system goes through each variation step by step using an algorithm, but over time it slows down more and more as the combinations increase. What starts off looking simple turns into something overwhelming, showing how even basic digital structures can become chaotic and impossible to fully complete.
